# 딸기 스마트팜 수익성 참고 (충청권 기준)

> 본 수치는 **공개된 업계/농촌진흥청 자료 기반 참고치**로, 실제는 입지·품종·판로에 따라 상이합니다. 사업 결정 전 지역 농업기술센터·선도농가 실견학 권장.

## 1. 매출 추정 (1동 = 약 200~300평)

| 항목 | 설향 (1기작 12월~5월) |
|---|---|
| 평당 수량 | 3.5~5 kg/평 |
| 평당 매출 | 35~60만원/평 (단가 10,000~15,000원/kg) |
| 300평 연매출 | **1.0~1.8억** (다수확·고급화 시 2억대) |

**관건**: 초기 출하 (11~12월) 단가가 kg당 2만원 넘어가는 시기가 있음 → 환경제어로 조기출하 시 매출 +30%

## 2. 초기 비용 (300평 1동 신규)

| 항목 | 비용 |
|---|---|
| 비닐하우스 (연동형) | 4,000~6,000만원 |
| 양액재배 고설베드 | 2,500~4,000만원 |
| 난방 (온풍기 + 지중난방) | 1,500~2,500만원 |
| 관수 시스템 | 300~500만원 |
| **스마트팜 자동화 (본 시스템 포함)** | **100~300만원** (센서+제어+서버) |
| 모종 | 200~400만원 (주당 400~600원 × 15,000주) |
| 기타 (양액, 잡자재) | 500만원 |
| **합계** | **약 9,000만원 ~ 1.4억** |

**정부지원**:
- ICT 융복합 확산사업: 시설비 50~60% 보조 (시군 농정과)
- 청년후계농 영농정착지원: 월 110만원 × 최대 3년
- 스마트농업 교육 후 선정시 추가 가산

## 3. 손익 (안정화 2~3년차 기준)

| 항목 | 300평 |
|---|---|
| 매출 | 1.4억 |
| 변동비 (양액·포장·공판수수료 등) | 3,500만원 |
| 고정비 (난방·감가·인건 일부) | 3,000만원 |
| **영농소득** | **7,500만원** |
| 순이익률 | 약 50% (스마트팜 도입 시 +5~10%p) |

## 4. 본 시스템의 매출 기여 포인트

| 기능 | 추정 효과 |
|---|---|
| 자동관수 (토양수분 기반) | 수확량 +5~8% (수분 스트레스 감소) |
| 야간 저온 알림/난방 | 동해 방지 → 전손 리스크 제거 |
| 환기 자동화 | 잿빛곰팡이병 발병 -30% (상품률 +3~5%) |
| 수확예측 (GDD) | 출하 스케줄링 → 최적 단가 시기 선점 |
| 판매/매출 DB | 공판장 vs 직거래 단가 실증 → 고단가 채널 전환 |

## 5. 실행 로드맵 (권장)

1. **Month 1**: 지역 농업기술센터 교육 + 선도농가 2~3곳 견학
2. **Month 2**: 작목반 가입, 공판장 출하 계약 점검
3. **Month 3~5**: 하우스 시공, 스마트팜 설치 (본 시스템)
4. **Month 6**: 정식 (보통 8~9월이 설향 정식기)
5. **Month 10~**: 수확 시작, 본 시스템의 자동제어 룰 현장 튜닝
